# WGFlutterDemo
跨平台，flutter技术总结：1、增加了一个完整项目案例demo（在Shop/flutter_shop_center下）；2、简单的页面局部 ；3、封装一些小的案例；4、简单易懂、帮助快速入手。5、每一个demo都可以单独运行。6、混编，与原生之间的交互，包括跳转，传值等
